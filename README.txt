使い方

1. ZIP を展開
2. 必要なライブラリをインストール
3. app.py があるフォルダで以下を実行

streamlit run app.py

主な仕様
- 直線のみ: これまでの再帰分割ロジック
- 直線＋道のり距離補正: まず直線で分割し、その後に境界町丁目だけを道のり距離で補正
- 1分割あたりの65歳以上人口 と 分割数 の両方を入力可能
- 採用する基準で、どちらを実行分割数に使うか切替可能

注意
- 道のり距離補正を使う場合は、public.chouchoumoku_route_matrix が事前に埋まっている必要があります
- 未登録ペアがある場合、アプリはエラー表示して停止します
